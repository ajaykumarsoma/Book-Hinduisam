    Contributing¶

    Thank you for your interest in improving this book! This is a living work that grows with community input.

    🤝 How You Can Help¶

    1. Report Errors or Suggest Improvements¶

    Found a typo, incorrect translation, or factual error?

    Open an Issue →

    Please include: - Chapter and section - What's wrong - Suggested correction (if applicable) - Source reference (for factual corrections)

    2. Suggest New Chapter Topics¶

    What should we write about next?

    Popular requests: - Specific deities (Gaṇeśa, Hanumān, Durgā) - Yoga practices and philosophy - Hindu festivals and their significance - Regional traditions - Modern reform movements

    Suggest a Topic →

    3. Add Citations or References¶

    Help us cite additional: - Primary Sanskrit texts - Traditional commentaries - Modern scholarly works - Archaeological findings

    Edit directly on GitHub or submit corrections via Issues.

    4. Improve Explanations¶

    If a concept is unclear or could be explained better:

    1. Fork the repository
    2. Edit the markdown file
    3. Submit a Pull Request with your improvements

    5. Add Images¶

    See Image Guide for how to add images to chapters.

    We need: - Temple photographs - Deity statues - Ritual ceremonies - Sanskrit manuscripts - Diagrams and illustrations

    6. Translate¶

    Help make this book accessible in: - Hindi (हिन्दी) - Tamil (தமிழ்) - Sanskrit (संस्कृत) - Bengali (বাংলা) - Other languages

    Translation workflow: 1. Create docs/[language-code]/ folder 2. Translate markdown files 3. Update mkdocs.yml with language navigation 4. Submit Pull Request

    📜 Contribution Guidelines¶

    Content Standards¶

    ✅ Do: - Cite primary sources (Vedas, Upaniṣads, etc.) - Use IAST for Sanskrit transliteration - Present multiple viewpoints (where applicable) - Maintain scholarly yet accessible tone - Add references and bibliography

    ❌ Don't: - Copy-paste from Wikipedia or other websites - Make unsourced claims - Insert personal religious beliefs as facts - Use offensive or disrespectful language - Oversimplify complex concepts without explanation

    Sanskrit Guidelines¶

    • Use IAST transliteration: ā, ī, ū, ṛ, ṃ, ḥ, ś, ṣ, ñ, etc.
    • First use: Full term + Devanagari + translation
    • Example: Prāṇa (प्राण) — life-force, vital energy
    • Subsequent uses: IAST only or English
    • Verify spellings using Sanskrit dictionaries

    Code of Conduct¶

    This is an inclusive, respectful community:

    ✅ Respectful dialogue
    ✅ Assume good faith
    ✅ Focus on ideas, not people
    ✅ Welcome diverse perspectives

    ❌ No harassment, hate speech, or personal attacks
    ❌ No proselytizing or religious debates
    ❌ No plagiarism

    🔧 Technical Contributions¶

    Setting Up Locally¶

    # Clone the repository
git clone https://github.com/ajaykumarsoma/Book-Hinduisam.git
cd Book-Hinduisam

# Install dependencies
pip install -r requirements.txt

# Serve locally
mkdocs serve
# Opens at http://127.0.0.1:8000

    File Structure¶

    docs/
├── index.md              # Home page
├── about.md              # About page
├── chapters/             # All chapters
│   ├── index.md
│   └── chapter_01_*.md
├── resources/            # Glossaries, bibliographies
└── stylesheets/          # Custom CSS

    Making Changes¶

    1. Fork the repository
    2. Create a branch: git checkout -b improve-chapter-1
    3. Make edits to markdown files
    4. Test locally: mkdocs serve
    5. Commit: git commit -m "Improve Sanskrit accuracy in Chapter 1"
    6. Push: git push origin improve-chapter-1
    7. Open Pull Request on GitHub

    📧 Contact¶

    • GitHub Issues: Technical problems, errors
    • GitHub Discussions: Ideas, questions, general feedback
    • Email: contribute@hinduism-book.org (if configured)

    Contributors will be acknowledged in the book's credits.
