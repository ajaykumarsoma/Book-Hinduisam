## 📜 Contribution Guidelines

### Content Standards

✅ **Do:**
- Cite primary sources (Vedas, Upaniṣads, etc.)
- Use IAST for Sanskrit transliteration
- Present multiple viewpoints (where applicable)
- Maintain scholarly yet accessible tone
- Add references and bibliography

❌ **Don't:**
- Copy-paste from Wikipedia or other websites
- Make unsourced claims
- Insert personal religious beliefs as facts
- Use offensive or disrespectful language
- Oversimplify complex concepts without explanation

---

### Sanskrit Guidelines

- **Use IAST transliteration:** ā, ī, ū, ṛ, ṃ, ḥ, ś, ṣ, ñ, etc.
- **First use:** Full term + Devanagari + translation
  - Example: *Prāṇa (प्राण) — life-force, vital energy*
- **Subsequent uses:** IAST only or English
- **Verify spellings** using Sanskrit dictionaries

## 🔧 Technical Contributions

### Setting Up Locally

```bash
# Clone the repository
git clone https://github.com/ajaykumarsoma/Book-Hinduisam.git
cd Book-Hinduisam

# Install dependencies
pip install -r requirements.txt

# Serve locally
mkdocs serve
# Opens at http://127.0.0.1:8000
```

### File Structure

```
docs/
├── index.md              # Home page
├── about.md              # About page
├── chapters/             # All chapters
│   ├── index.md
│   └── chapter_01_*.md
├── resources/            # Glossaries, bibliographies
└── stylesheets/          # Custom CSS
```

### Making Changes

1. **Fork** the repository
2. **Create a branch:** `git checkout -b improve-chapter-1`
3. **Make edits** to markdown files
4. **Test locally:** `mkdocs serve`
5. **Commit:** `git commit -m "Improve Sanskrit accuracy in Chapter 1"`
6. **Push:** `git push origin improve-chapter-1`
7. **Open Pull Request** on GitHub
